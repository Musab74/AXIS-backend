[현재 프로세스 문제점]

• 총 소요: 약 2시간/주

[자동화 목표]
총 소요시간을 30분 이내로 단축하되, 품질을 유지하세요.

[자동화에 활용할 데이터]

Step3. 코멘트 생성: AI에 전주 대비 변화 분석+코멘트 요청 (AI: 분석+코멘트 / 사람: 검토)

[AI 실행 결과] 위 프롬프트를 실행하여 생성된 회의록 산출물: "일시: 2026.07.02 / 참석자: 김팀장, 이대리, 박사원 / 안건1: 하반기 예산 → 결정: 10% 증액 → 액션: 이대리, 7/10까지 예산안 수정"
[기대 효과] 75분→15분 (80% 절감). 표준 형식 자동 적용으로 회의록 일관성 확보.
